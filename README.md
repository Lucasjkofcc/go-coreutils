# Go Coreutils

Uma recriação (em desenvolvimento) do GNU Coreutils feita em Go.

## Comandos implementados

- basename
- cat
- cp
- date
- dirname
- echo
- false
- head
- ls
- mkdir
- mv
- printf
- pwd
- rm
- rmdir
- seq
- sleep
- tail
- touch
- true
- wc
- which
- yes

> Projeto em desenvolvimento. Nem todos os comandos possuem todas as funcionalidades do GNU Coreutils.
